This is a guide to creating your first Flutter app. In this article, I'm going to guide you to create your first Flutter application on Android Studio and run the app with Android Emulator successfully. This tutorial is using Flutter 3.0.5 Install Android Studio Download: [https://developer.android.com/studio](https://developer.android.com/studio "new tab") Install Flutter Plugin Install Dart Plugin After installation, please restart the Android Studio. New Flutter Project Set Flutter SDK path After setting, click "Next". Project Settings After setting, click "Finish". Device Manage Click "Device Manager" Create device Set the simulator definition Start the device Open the window of the emulator Menu bar > View > Tool Windows > Emulator Emulator SDK Manager Click SDK Manager, and install the Android SDK version you want. Set Project Structure Set SDK Apply and OK Click Run button Finish Keep learning here, we have a lot to teach.
