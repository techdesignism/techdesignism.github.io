Arrow function The arrow function allows us to create a simplified function consisting of a single expression. We can omit the curly brackets and the return keyword. void main() { int age = getAge(); String name = getName(); print( name + ' ' + age.toString() ); print( '$name is ${age.toString()} years old.' ); //string interpolation } int getAge() => 25; String getName() => 'Mary'; Output: Mary 25 Mary is 25 years old. List, add and remove The List.remove( ) method in dart is used to remove the first item from the list. As an argument, this method takes the value which is going to be removed. It returns true if the item is present in the list and successfully removed and returns false if the item is not present in the List. This method reduces the List length size by one. If the value you passed to remove is not present in the list then there is no modification in List will be reflected but the method will return false. void main() { List names = ['Ben','Alex']; print(names); names.add('Mary'); print(names); names.add(56531231); print(names); names.remove('Ben'); print(names); names.add('May'); names.add('Ben'); names.add('May'); print(names); names.remove('May'); print(names); } Output: [Ben, Alex] [Ben, Alex, Mary] [Ben, Alex, Mary, 56531231] [Alex, Mary, 56531231] [Alex, Mary, 56531231, May, Ben, May] [Alex, Mary, 56531231, Ben, May] List of a specific type These codes have errors because the list is specified the data type as String but the codes insert an int value. void main() { List<String> names = ['Ben','Alex',30]; print(names); } Output: Error: A value of type 'int' can't be assigned to a variable of type 'String'.
