This tutorial is using Flutter 3.0.5 Simplify the demo app First, simplify the demo app like the one below. import 'package:flutter/material.dart'; void main() { runApp(const MyApp()); } class MyApp extends StatelessWidget { const MyApp({Key? key}) : super(key: key); @override Widget build(BuildContext context) { return MaterialApp( theme: ThemeData( primarySwatch: Colors.green, ), home: Scaffold( appBar: AppBar( title: Text('App Bar Title'), ), body: Center( child: Text('Content'), ), floatingActionButton: FloatingActionButton( onPressed: (){}, child: const Icon(Icons.add), ), ) ); } } We can see there are 3 properties of Scaffold: appBar, body, and floatingActionButton. We can use appBar to control the top bar. body is the content below the top bar. floatingActionButton is the right bottom corner circle button.
